- Check the Python version

- Check whether all required packages work correctly

## Step 2: Dataset Preparation

- Download the required dataset

- Check the dataset folder structure
- Verify all client folders
- Check normal, abnormal and testing data

## Step 3: Source Code Understanding

- Identify the data-loading code

- Check the configuration files

## Step 4: Initial Test

- Run the program with a small configuration
- Check whether the dataset loads correctly
- Check whether clients start correctly
- Save the terminal output
- Take screenshots of the results

## Step 5: Full Experiment

- Run the complete experiment
- Record training accuracy and loss
- Save generated graphs and results
- Compare the result with the original research paper
